Ejemplos del proyecto Albora

  • indexjquery.html
    Ejemplo de uso básico de jQuery: cambia el color de un texto, muestra la longitud de un input en tiempo real y saluda al enviar el formulario.
  • indexform.html
    Formulario HTML con validación y manejo de eventos. Tratamiento de información en entorno cliente. Prevent default.
  • indexrecursivo.html
    Ejemplo de función recursiva en JavaScript.
  • closures.html
    Ejemplo básico de closures en JavaScript.
  • closures2.html
    Ejemplo avanzado de closures con funciones que recuerdan su contexto.
  • oo.html
    Ejemplo de programación orientada a objetos en JavaScript.
  • llamadaajax.html
    Ejemplo de doble llamada AJAX para obtener datos de un servidor.
  • callback1.html
    Ejemplo de llamada a función callback simple.
  • callbackhell.html
    Demostración de callback hell con múltiples funciones anidadas. Resultado no ordenado.
  • promesas1jquery.html
    Intro a las promesas con obhjeto Deferred.
  • promesassencillo.html
    Ejemplo sencillo de uso de promesas en JavaScript para obtener varios usuarios simulando retrasos.
  • promesasjquery.html
    Ejemplo de callbacks anidados pero ordenados con promesas.
  • promesasES6.html
    Ejemplo de callbacks anidados pero ordenados con promesas utilizando sólo ES6.
  • promesasES6total.html
    Extensión de promesasES6: para cada post de cada autor muestra también los comentarios del post.
  • promesasES6funcional.html
    Versión funcional que muestra usuarios, posts y comentarios usando map/filter/reduce.
  • asyncawaiteasy.html
    Introducción a async/await para llamadas AJAX ordenadas.
  • asyncawait.html
    Misma funcionalidad que promesasES6total pero implementada con async/await y promesas.
  • asyncawaitfunctional.html
    Versión async/await en estilo funcional que muestra usuarios, posts y comentarios.
  • localstorageES6.html
    Ejemplo simple de persistencia en localStorage: persistir valores desde un input y resetearlos.
